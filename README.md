# المنيو الإلكتروني (HTML + CSS + JS)

- واجهة RTL عربية جميلة لعرض:
  - اسم المتجر وشعاره
  - سلايدر للإعلانات من جدول `ads_banners`
  - قائمة الأصناف من جدول `categories`
  - شبكة المنتجات حسب كل صنف من جدول `products`

## الإعداد
1) افتح ملف `config.js` واملأ:
   - `SUPABASE_URL`
   - `SUPABASE_ANON_KEY`

> ملاحظة: تم تفعيل RLS مع سياسات قراءة عامة للجدوال المطلوبة في المخطط المرفق، لذا يمكن القراءة بالمفتاح العام (anon) فقط.

2) ارفع البيانات إلى جداولك (stores, ads_banners, categories, products). سيتم جلب أول صف من `stores` لاسم/شعار المتجر.

3) افتح `index.html` في المتصفح.

## ملاحظات
- ترتيب الأصناف يعتمد على `sort_order` ثم `name`.
- السلايدر يعرض الإعلانات النشطة فقط (`is_active = true`).
- شبكة المنتجات تعرض المتاح أولًا، وتعرض وسم "غير متوفر" للنفاد.
- تصميم متجاوب وخفيف.
